# 登录界面重新设计 V2

## 背景

第一版蓝色玻璃拟态方案被否：太蓝、太丑、卡片与边缘留白过大。本版重新设计。

## 核心约束

- **上下左右最多 8px 留白**：卡片几乎占满整个视口
- **告别蓝色**：全灰白色系，干净清爽
- **非对称布局**：左侧浅灰品牌面板 + 右侧纯白表单区
- **几何色块装饰**：同心圆静态图形，无动画

## 布局方案

```
┌─────────────────────────────────────────────────┐ 8px 灰色底 (#f8f9fa)
│  ┌──────────┬──────────────────────────────────┐│ 白色卡片 12px圆角
│  │          │                                  ││
│  │  品牌图标│         欢迎回来                  ││
│  │  本体建模│         登录以访问您的工作台       ││
│  │  ONTOLOGY│                                  ││
│  │          │    [用户名输入框]                 ││
│  │  ◎ 同心圆│    [密码输入框]                   ││
│  │  几何装饰 │    [登  录]                      ││
│  │          │                                  ││
│  │  38%     │              62%                 ││
│  └──────────┴──────────────────────────────────┘│
└─────────────────────────────────────────────────┘
```

- 外层：`background: #f8f9fa; padding: 8px; width: 100vw; height: 100vh`
- 内层卡片：`background: #fff; border-radius: 12px; display: flex; height: 100%`
- 左面板：`flex: 0 0 38%; background: #f4f6f8`
- 右面板：`flex: 1; background: #fff`

## 视觉层次

### 左面板
- 品牌图标：40px 深灰方块（#1e293b），圆角 10px，内含白色 SVG 网络图标
- 标题："本体建模系统"，20px / 800 weight / #1e293b
- 副标题："ONTOLOGY MODELING SYSTEM"，8px / 600 weight / #94a3b8 / 宽字间距
- 几何装饰：三个同心圆，位于品牌区下方
  - 外圆：100px，2px 边框，#d1d5db
  - 中圆：60px，1.5px 边框，#e5e7eb
  - 内圆：28px，填充 #1e293b，不透明度 0.08

### 右面板
- 标题："欢迎回来"，18px / 700 weight / #1e293b
- 副标题："登录以访问您的工作台"，13px / #64748b
- 表单宽度：300px，在右侧面板居中
- 输入框：padding 12px 14px，边框 1.5px #e2e8f0，背景 #f8fafc，圆角 8px
- 聚焦态：边框 #94a3b8，box-shadow 0 0 0 3px rgba(148,163,184,0.15)
- 登录按钮：全宽，背景 #1e293b，文字白色，padding 13px，圆角 8px
- hover：背景 #0f172a，translateY(-1px)
- 错误提示：红色文字 + 浅红底

## 响应式

- `> 768px`：左右分栏 38%/62%
- `≤ 768px`：上下堆叠，左面板变为顶部横条（高度 auto），右面板填充剩余空间
- `≤ 480px`：8px 外层保留，内部 padding 缩小，左面板进一步压缩

## 实施范围

- **修改文件**：`frontend/src/components/auth/LoginView.vue`
  - 重写 `<template>`：外层 8px padding + 左右分栏
  - 重写 `<style>`：灰白配色 + 几何装饰 + 表单样式
  - 保留 `<script setup>` 登录逻辑不变

## 技术约束

- 无 CSS 动画（纯静态）
- 移除 Google Fonts 引入（使用系统字体栈）
- 登录 API 调用逻辑完全不变
- 保留 `prefers-reduced-motion`（可选）
