## 目标
把左侧“折叠式按钮区”改为“分区 + 页签”的非折叠式工作台；同时保持顶部三视角（概览/数据映射/本体推理）不变。新建项目后默认跳转到“数据导入”页签（不强制）。

## 范围
- 左侧改为 3 个页签：项目 / 数据导入 / 建模
- “建模”页签中提供两颗主按钮：新增节点、新增关系
- 新增关系支持两种方式：画布点选 与 弹窗选择（可搜索）
- 数据导入复用现有 DataSourcePanel（上传/数据库/预览导入）
- 不调整顶部三视角结构与其路由/接口

## 信息架构
### 顶部（保持不变）
- 概览（leader）
- 数据映射（engineer）
- 本体推理（process：规则/推理/互斥等）

### 左侧（工作台页签）
#### 1) 项目
- 项目选择/搜索
- 新建/重命名/删除
- 新建成功后：默认切到“数据导入”页签

#### 2) 数据导入
- 上传文件（Excel/CSV）
- 连接数据库（达梦/模拟）
- 预览并一键导入生成本体网络
- 导入成功后：建议自动切到 process 视角以便立即查看画布（不改变左侧页签状态）

#### 3) 建模
- 新增节点：弹窗输入名称 → 自动放置在可视区中心附近
- 新增关系：两种模式均支持
  - 画布点选（默认）：点源节点 → 点目标节点 → 填写关系名/权重确认
  - 弹窗选择：源/目标节点下拉（可搜索）+ 关系名/权重确认
- 保存/清空等管理动作归并到此页（不折叠）

## 交互细节
### 新建项目默认跳转
- 新建项目成功后：左侧页签切换到“数据导入”
- 顶部视角不强制切换；数据导入完成后可切换到 process（推荐默认切换）

### 新增关系双模式
- “新增关系”按钮旁提供一个轻量开关：画布点选 / 弹窗选择
- 记忆用户上次选择（本地存储）
- 画布点选可通过 Esc 取消

## 状态与数据流
- 左侧页签状态独立于顶部视角状态
- 项目列表在新建/删除后必须即时刷新并保持 currentProjectId 一致
- 204 No Content 的删除接口需被前端正确处理（避免 UI 进入异常态）

## 验收标准
- 左侧不再出现折叠式（accordion）交互
- 任意顶部视角下，新建/删除项目均可使用且有即时反馈
- 新建项目后默认切到“数据导入”页签
- “建模”页签中可直接新增节点/新增关系（两种方式均可用）
