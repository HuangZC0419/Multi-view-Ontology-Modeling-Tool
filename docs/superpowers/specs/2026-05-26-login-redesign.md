# 登录界面重新设计

## 背景

当前登录界面（`LoginView.vue`）存在三个问题：
1. 左侧品牌文字偏底部，未居中
2. 左侧动态知识图谱（SVG 圆圈 + 连线动画）视觉效果不佳
3. 右侧登录表单区域大面积留白（340px 表单在 52% 宽度面板中）

用户期望整体界面有"突出的感觉"，给人留下深刻印象。

## 设计方向

**蓝色玻璃拟态 (Blue Glassmorphism) + 全屏居中宽卡片**

- 配色完全沿用系统现有蓝色调（#1E40AF, #2563eb, #3b82f6, #60a5fa）
- 移除左右分栏，改为全屏蓝色渐变背景
- 一张居中玻璃拟态宽卡片承载品牌信息 + 登录表单
- 背景用柔和光晕 + 浮动粒子替代原来生硬的 SVG 圆圈连线

## 布局方案

```
┌─────────────────────────────────────────────────┐
│                                                 │
│          ● 浮动光点    ○ 大光晕 (呼吸)            │
│                                                 │
│    ┌───────────────────────────────────────┐    │
│    │  glass card (540px, blur 24px)        │    │
│    │  ┌──────────┐  ┌───────────────────┐  │    │
│    │  │ 品牌图标  │  │  用户名输入框      │  │    │
│    │  │ 本体建模  │  │  密码输入框        │  │    │
│    │  │ ONTOLOGY │  │  [登  录] 按钮     │  │    │
│    │  └──────────┘  └───────────────────┘  │    │
│    └───────────────────────────────────────┘    │
│                                                 │
│          ○ 小光晕      ● 粒子                    │
│                                                 │
└─────────────────────────────────────────────────┘
```

- 卡片宽度：540px，`max-width: 92vw`（小屏自适应）
- 卡片内左右双栏：左侧品牌区（固定宽），右侧表单区（弹性）
- 小屏（<640px）退化为上下堆叠

## 视觉层次

### 背景层
- 深蓝→亮蓝多层渐变（#0f2744 → #1a3a6b → #1e40af → #2560d0 → #3075f0）
- 3 个半透明径向渐变光晕球体，带 CSS 呼吸动画（scale/opacity 循环）
- 6 个浮动光点粒子，CSS animation 缓慢游动（translate + opacity）

### 卡片层
- `background: rgba(255,255,255,0.1)` + `backdrop-filter: blur(24px)`
- `border: 1px solid rgba(255,255,255,0.18)`
- `border-radius: 24px`
- `box-shadow`: 外部投影 + inset 微光边框

### 品牌区（卡片左侧）
- 品牌图标：蓝色渐变圆角方块 + SVG 网络图标
- 标题："本体建模系统"，白色，22px，800 weight
- 副标题："ONTOLOGY MODELING SYSTEM"，半透明白色，9px，宽字间距

### 表单区（卡片右侧）
- 标签：半透明白色，12px，600 weight
- 输入框：`rgba(255,255,255,0.08)` 背景 + `rgba(255,255,255,0.12)` 边框，圆角 10px
- 聚焦态：边框变亮 + 微弱外发光
- 登录按钮：`rgba(255,255,255,0.9)` 纯白高对比，hover 变全白 + 上浮 1px
- 错误提示：红色半透明背景条
- 加载态：白色 spinner

## 动画规格

| 元素 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 光晕球体 1 | scale 1→1.08→1, opacity 循环 | 8s | ease-in-out infinite |
| 光晕球体 2 | scale 1.05→0.95→1.05 | 6s | ease-in-out infinite |
| 光晕球体 3 | translate 缓慢漂移 | 12s | ease-in-out infinite alternate |
| 浮动粒子 | translate X/Y 各自独立周期 | 4-10s 各不同 | ease-in-out infinite alternate |
| 卡片 | 入场 opacity 0→1 + translateY 20px→0 | 0.6s | ease-out |

## 响应式

- `> 640px`：宽卡片 540px，左右双栏
- `≤ 640px`：卡片宽度 92vw，内部上下堆叠，品牌区缩小
- `≤ 400px`：卡片 padding 缩小，品牌图标缩小

## 实施范围

- **修改文件**：`frontend/src/components/auth/LoginView.vue`
  - 重写 `<template>`：移除左右分栏，改为居中卡片布局
  - 重写 `<style>`：蓝黑渐变背景 + 玻璃拟态卡片 + 光晕/粒子动画
  - 保留 `<script setup>` 登录逻辑不变（API 调用、表单验证、emit 事件）
- **不修改**：`App.vue`、后端代码、其他组件

## 技术约束

- 纯 CSS 动画，不引入额外依赖
- 保留 `prefers-reduced-motion` 尊重（动画降级为静态）
- 保留 `Plus Jakarta Sans` 字体引入
- 登录 API 调用逻辑完全不变
