# 侧边栏折叠面板 + 二部图数据映射 + OCR 清理 设计文档

日期: 2026-05-23 | 状态: 已确认

## 一、侧边栏折叠面板

### 结构（管理者）
- 画布编辑（折叠）— 新增本体、保存、清空
- 关系与规则（折叠）— 连线模式、子本体模式、规则设置
- 因果分析（展开）— 贝叶斯分析、权重配置
- 数据导入（折叠）— DataSourcePanel

### 结构（普通用户）
- 因果分析（展开）— 贝叶斯分析、权重配置（只读）

### 交互规则
- 同时只展开一组
- 组标题用 SVG 图标
- 过渡动画 200ms

## 二、数据映射 → 二部图

- 左列：本体概念（按业务域分组）
- 右列：数据源·表名（达梦紫/Excel绿/CSV橙）
- 中间：字段名连线标签
- 固定布局，无物理模拟
- 缩放平移，按域折叠

## 三、OCR 清理

- 后端：删除 RapidOCR/jieba 导入、OCR 路由、OCR 函数
- 前端：删除 OCR 变量/函数/模板
